# WEAR Work

This repository is assignment by wear.

## How to use

### 0. Install golang

Install golang.  
See [Golang](https://golang.org/doc/install).

### 1. Clone this repository

```bash
$ git clone https://github.com/phyyou/wearwork.git
$ cd wearwork
```

### 2. Run

```bash
$ go mod tidy
```

```bash
$ go run main.go
```

Also, you can run it with repl mode.

```bash
$ go run main.go --repl
```
